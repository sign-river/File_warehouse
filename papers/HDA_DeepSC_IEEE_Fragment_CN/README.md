# HDA-DeepSC Chinese IEEE-Style Fragment

This directory contains an editable Chinese academic paper fragment integrating:

- imperfect-CSI receiver experiments;
- uncertainty-aware MAP soft demodulation;
- BER/PER/FER evaluation;
- artificial physical packet erasure;
- semantic feature-packet erasure;
- digital/analog branch weight control;
- operator-facing interactive experiment design.

## Generated deliverables

The GitHub Actions workflow produces:

- `HDA_DeepSC_IEEE_Fragment_CN.pdf`: IEEE-style two-column PDF compiled with XeLaTeX;
- `HDA_DeepSC_IEEE_Fragment_CN.docx`: editable Chinese Word draft with native Word equations generated by Pandoc;
- `HDA_DeepSC_IEEE_Fragment_CN_Source.zip`: source package;
- `build_report.txt`: PDF metadata and font report.

## Local build

```bash
python -m pip install matplotlib numpy python-docx
python generate_figures.py
python make_reference_docx.py
latexmk -xelatex -interaction=nonstopmode -halt-on-error fragment.tex
pandoc fragment_word.md \
  --from=markdown+tex_math_dollars \
  --citeproc \
  --bibliography=references.bib \
  --reference-doc=reference.docx \
  --resource-path=. \
  -o HDA_DeepSC_IEEE_Fragment_CN.docx
```

The Word version is intentionally single-column and editing-friendly. The PDF uses the IEEE conference two-column layout. Numerical claims are limited to the previously available controlled-NMSE results; artificial erasure and recovery results are left as an experimental protocol rather than fabricated performance claims.
